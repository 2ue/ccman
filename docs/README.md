# 文档导航（精简版）

本目录仅保留**可直接指导使用/排障**的文档。设计评审/过程总结类文档已清理，以避免重复与过时信息。

## 推荐阅读顺序

1. 项目总览与命令示例：`README.md`（仓库根目录）
2. 需要理解写入逻辑/排查配置：`docs/工具配置原理.md`
3. ccman 不可用时的手动配置：
   - `docs/如何手动配置-Codex.md`
   - `docs/如何手动配置-Claude-Code.md`
   - `docs/如何手动配置-Gemini-CLI.md`
4. 官方配置项参考（离线备份）：
   - `docs/官方文档/codex/config.md`
   - `docs/官方文档/claude/settings.md`
   - `docs/官方文档/gemini/config.md`
5. GMN 一键脚本（可选）：`scripts/README-GMN-SETUP.md`
